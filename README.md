# Shifuairesearch Tap

## How do I install these formulae?

`brew install shifuairesearch/tap/<formula>`

Or `brew tap shifuairesearch/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "shifuairesearch/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
